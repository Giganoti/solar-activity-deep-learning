# Report

This folder contains the final Master's Thesis report for the project **Deep Learning for Extreme Solar Activity Prediction**.

The thesis focuses on the prediction of extreme solar activity using satellite observations and deep learning models. The study analyses two main solar phenomena: solar flares and coronal mass ejections (CMEs).

The work combines data from GOES-18, SDO/HMI-SHARP and SOHO/LASCO, and compares deep learning architectures such as 1D CNN and LSTM models.

The full report is written in Spanish.
