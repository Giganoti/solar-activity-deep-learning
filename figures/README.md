# Figures

This folder contains selected figures from the Master's Thesis project **Deep Learning for Extreme Solar Activity Prediction**.

The figures provide a visual summary of the main parts of the project: the satellite data sources, the X-ray flux used for solar flare prediction, and the performance of the deep learning models applied to solar flares and coronal mass ejections.

Only selected figures are included in this repository in order to give a clear overview of the work without duplicating the full thesis report.

## Selected figures

* `fig04_satellite_data_sources.png`
  Overview of the satellite missions used in the project: SDO, SOHO and GOES. These missions provided the observational basis for the flare and CME prediction tasks.

* `fig05_goes_xray_flux_peaks.png`
  X-ray flux time series from GOES, showing the peaks associated with solar flare activity. This signal was used as the main input for the solar flare prediction task.

* `fig06_flare_cnn_precision_recall.png`
  Precision-recall curve for the 1D CNN model applied to solar flare prediction.

* `fig07_flare_cnn_predictions.png`
  Comparison between real solar flare events and the predictions generated by the CNN model.

* `fig08_flare_cnn_metrics.png`
  Summary of the main evaluation metrics obtained by the CNN model in the solar flare prediction task.

* `fig10_flare_lstm_metrics.png`
  Evaluation metrics obtained by the LSTM model for solar flare prediction, allowing comparison with the CNN approach.

* `fig11_cme_cnn_results.png`
  Results obtained by the CNN model for coronal mass ejection prediction. These results reflect the greater difficulty of predicting CMEs using the available input variables.

* `fig13_cme_lstm_results.png`
  Results obtained by the LSTM model for CME prediction, used to compare the behaviour of recurrent and convolutional architectures in this second prediction task.

## Notes

The selected figures show that the prediction of solar flares achieved more promising results than the prediction of CMEs. This difference is one of the main conclusions of the project: X-ray flux data contain clearer temporal patterns before solar flare events, while CME prediction likely requires additional physical context and more complete coronal information.
